Detailed Description

An abstract base class for transition parameters, which allow intermediate generation of steps in a sequence. If used, the idea is that the transition parameters can work as follows: (s, a) => T => (o, r, s').

This is useful when model changes need to be applied, as the transition parameters can store values such as sampled random numbers or intermediate calculations, allowing the transitions to be recalculated more efficiently. Also, not having to re-randomize means that unnecessary divergence of histories due to resampling can be avoided.
